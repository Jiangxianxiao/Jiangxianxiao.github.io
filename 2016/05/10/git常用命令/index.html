git常用命令

2016-05-10 09:08:14
  1. 创建版本库 git init
  2. 添加文件到版本库 git add
  3. 提交文件 git commit -m <说明>
  4. 查看状态 git status
  5. 查看比较结果 git diff
  6. 查看历史记录 git log
  7. 查看历史命令 git reflog
  8. 丢弃工作区修改 git checkout –file
  9. 丢弃缓存区添加 git reset HEAD file
  10. 从版本库删除 git rm
  11. 推送当前分支致远程 git push [-u 关联本地与远程分支] [远程分支] [本地分支]
  12. 关联远程库 git remote add origin git@server-name:path/repo-name.git
  13. 克隆远程库 git clone
  14. 分支:
    • 查看分支 git branch
    • 创建分支 git branch
    • 切换分支 git checkout
    • 创建并切换分支 git checkout -b
    • 合并分支到当前分支 git merge
    • 删除分支 git branch -d
    • 强行删除 git branch -D
  15. 查看分支合并图 git log –graph
  16. 暂停:
    • 暂存工作空间 git stash
    • 取回工作空间 git stash pop
  17. 远程库操作:
    • 查看信息 git remote -v
    • 抓取分支 git pull
    • 本地创建对应的远程分支 git checkout -b branch-name origin/branch-name
    • 建立本地分支与远程关联 git branch –set-upstream branch-name origin/branch-name
  18. 标签:
    • 查看 git tag
    • 打标签 git tag
    • 查看标签信息 git show
    • 推送远程标签 git push origin
    • 删除远程标签 git push origin :refs/tags/
