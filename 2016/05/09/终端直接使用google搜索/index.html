终端直接使用google搜索

2016-05-09 23:54:34

 最近在使用终端的过程中时常有google需求，而打开浏览器搜索显然比较繁琐，本着程序员偷懒的原则，想直接在终端发起搜索，利用xdg-open，实现起来很简单：

  • 在已添加至PATH的目录，如/user/bin/，下新建文件google

  • 输入代码：
    #!/bin/bash
    if [ ! -n “$1” ]; then
    xdg-open http://www.google.com.hk
    fi

    if [ -n $1 ] ; then
    xdg-open http://www.google.com.hk/search?q=$1;
    fi

  • chmod +x添加可执行权限

 这样就可以实现终端搜索了，比如输入 google 英超，浏览器自动打开并执行搜索结果，同理也可以设置baidu、bing等。
